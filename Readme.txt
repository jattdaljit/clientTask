Note : Make sure that maven for eclipse is installed!

Follow the below mentioned steps to add project to eclipse:

1. Open Eclipse
2. Go to File > Import
3. Search for Maven > Existing Maven Project
4. Browse and choose the code folder. 
5. Check the pom.xml under projects
6. Click Finish

Follow the below mentioned steps to build the project:

1. Right Click on added project in Project Explorer.
2. Go to Run As > Maven install

Follow the below mentioned steps to run all test cases at once:

1. Expand the current project.
2. Right click on src/test/java
3. Go to Run As > JUnit Test

Assumptions:
1. Assuming that all the conditions are being checked.
2. Using POST method while calling url.
